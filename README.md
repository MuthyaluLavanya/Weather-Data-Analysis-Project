# Weather Data Analysis using Python

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a weather dataset using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Tasks
- Data Exploration
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Correlation Analysis

## Visualizations
- Histogram
- Count Plot
- Scatter Plot
- Bar Plot
- Box Plot
- Correlation Heatmap
- Pair Plot

## Key Insights
- Analyzed temperature distribution.
- Identified the most frequent weather conditions.
- Studied the relationship between temperature and humidity.
- Compared average temperature across weather conditions.
- Explored correlations among numerical variables.
